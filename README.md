# PythonEdu2

Flask와 RestX를 사용하여 api 서버를 개발합니다.

```
// 파이썬 라이브러리 설치 방법
pip install <lib-name>

// 종속성을 관리하기 위한 txt 파일 생성
pip freeze > requirements.txt

// requirement.txt파일로 관련된 종속성들을 설치
pip install -r requirements.txt

// Zappa 설치
pip install zappa

// Zappa 설정
zappa_settings.json 파일 생성 후 정보 기재

// Zappa 배포
zappa deploy dev
```
